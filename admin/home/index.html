Справка/Руководство пользователя

Добавление новостей

Для добавления нового материала в разделе «Новости» необходимо нажать кнопку «Добавить новость». Необходимо заполнить поля «Заголовок», «Текс новости», Для добавления видео необходимо скопировать ссылку с www.youtube.com и вставить в поле «Ссылка на YouTube». Для добавления изображения не обходимо нажать кнопку «Выбрать файл» и выбрать изображение с компьютера.

Управление слайдером

Фотографии для слайдера должны быть размером 1200*300 (Ширина*Высота). Для изменения или удаления фотографии на слайдере необходимо перейти на в меню «Слайдер». Выбрать нужный слайд с помощью кнопки «Редактировать». Для загрузки фотографии необходимо нажать «Выбрать файл» и нажать кнопку сохранить.

Тренеры

Добавления новой записи для тренера. В меню «Тренеры» использовать кнопку «Добавить тренера». Заполнить необходимые поля «ФИО», «Биография», телефон в поле «Контакты», добавлении фотографии выполняется кнопкой «Выбрать файл». Что бы сохранить новую запись нажимаем кнопку сохранить. Редактирование существующих записей делается кнопкой «Редактировать».

Спортсмены

На странице «Спортмены» выполняется редактирование и добавление записей о спортсменах. Аналогично «Тренерам»

Расписание

На странице «Расписание» возможно добавлять или редактировать расписание занятий.

Мероприятия

На странице «Мероприятия» можно редактировать уже существующие мероприятия или добавить новые. Так же по необходимости можно перенести в архивы прошедшие мероприятия.

Документы

На странице «Документы» можно добавлять или удалять документы

Галерея

На странице «Галерея» можно управлять фотогалереей на сайте. Для создания нового альбома необходимо нажать «Добавить альбом» - вводим название альбома и добавляем обложку с помощью кнопки «Выбрать файл».

Добавление фотографий. Необходимо выбрать нужный альбом и нажать кнопку «Редактировать». Для добавления фотографий используйте кнопку «выбрать файл».

Пользователи

На странице «Галерея» служить для добавления пользователей с необходимыми правами.
